Demo Xpath

To easily get the xpath of an element in Chrome. Click the element, right click Inspect. Then in the DevTools, right click the HTML element, then Copy > Copy xpath.

Example Tests

-
  set: blogPostTitle
  value: '(//*[contains(@class, "block-title")])[3]'

-
  visit: /test_subject.html
  find:
    -
      why: XPath can also be used to locate things, so if you want to be sure that the third .block-title is a given value, you should use xpath.
      xpath: '(//*[contains(@class, "block-title")])[3]'
      text: Latest Blog Post
    -
      xpath: '(//*[contains(@class, "block-title")])[3]'
      is: Latest Blog Post
    -
      why: Assert we can use variables in the selector.
      xpath: ${blogPostTitle}
      is: Latest Blog Post
    -
      xpath: '(//*[contains(@class, "block-title")])[3]'
      is not: Blog Post
    -
      xpath: '(//*[contains(@class, "block-title")])[3]'
      contains: Blog Post
    -
      xpath: '(//*[contains(@class, "block-title")])[3]'
      matches: /blog post/i

Provided by the xpath handler.
