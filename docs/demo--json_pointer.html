JSON Pointer

Implements JSON Pointer to be able to extract data from JSON responses.

Features:

  • Will only work on valid JSON responses. A valid response is one that explicitly has content-type: application/json whose body can be parsed as JSON without error.

More Links

  • https://packagist.org/packages/php-jsonpointer/php-jsonpointer

Example Tests

-
  why: Demonstrate the Json Handler syntax.
  url: /test_subject.php?q=json_pointer__luke.json
  find:
    -
      pointer: /height
      is: 172
    -
      pointer: /hair_color
      set: hairColor
    -
      pointer: /skin_color
      contains: air
    -
      pointer: /mass
      not contains: 6
    -
      pointer: /hair_color
      is: ${hairColor}
    -
      pointer: /height
      is not: 200
    -
      pointer: /homeworld
      matches: /^https:/
    -
      pointer: /homeworld
      not matches: /^https:\/\/swapi\.dev\/api\/films\//
    -
      pointer: /films
      count: 4

-
  why: Demonstrate the Json Handler syntax with nested data.
  url: /test_subject.php?q=json_pointer__nested.json
  find:
    -
      pointer: /data/0/categories/0/CategoryID
      is: 1
    -
      pointer: /messages
      count: 0
    -
      pointer: /data/0/age
      is:
    -
      pointer: /data/0/age
      is: ''
    -
      pointer: /data/0/age
      is: null
    -
      pointer: /data/1/age
      is:
    -
      pointer: /data/1/age
      is: ''
    -
      pointer: /data/1/age
      is: null

Provided by the json_pointer handler.
