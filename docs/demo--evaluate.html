Evaluate

Allows runtime evaluation such as comparing values captured using set.

https://symfony.com/doc/current/components/expression_language.html

You may need to wrap some values in quotes, e.g. "2022-09-07T20:04:04+0000" < "2022-09-07T20:04:06+0000".

Example Tests

# This test also tests to make sure that variables can be reassigned across tests.
-
  why: Demonstrate how to assert an image's aspect ratio using `eval`.
  visit: /test_subject.html
  find:
    -
      dom: img.tree
      style: width
      set: width
    -
      dom: img.tree
      style: height
      set: height
    -
      why: Assert image has a landscape aspect ratio.
      eval: ${height} < ${width}
-
  why: Assert a different image this time.
  visit: /test_subject.html
  find:
    -
      dom: img.lake
      style: width
      set: width
    -
      dom: img.lake
      style: height
      set: height
    -
      why: Assert image has a tall aspect ratio.
      eval: ${height} > ${width}
-
  why: Demonstrate eval at the test level
  eval: ${height} > ${width}

Provided by the evaluate plugin.
