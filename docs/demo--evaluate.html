Evaluate

Allows runtime evaluation such as comparing values captured using set.

https://symfony.com/doc/current/components/expression_language.html

Example Tests

# This test also tests to make sure that variables can be reassigned across tests.
-
  why: Demonstrate how to assert an image's aspect ratio using `eval`.
  visit: /test_subject.html
  find:
    -
      dom: img.tree
      style: width
      set: width
    -
      dom: img.tree
      style: height
      set: height
    -
      why: Assert image has a landscape aspect ratio.
      eval: ${height} < ${width}
